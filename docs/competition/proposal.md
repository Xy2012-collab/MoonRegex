## 项目简介

MoonRegex 是一个纯 MoonBit 实现的正则表达式引擎，基于 AST 编译 + 递归回溯匹配。支持完整的正则语法，包括字面量、通配符、量词、分支、字符类、范围、转义、计数重复等。提供 match、find、find_all、replace、split、count、match_result 七种 API。零外部依赖。

## 核心功能范围

### 正则语法
- 字面量匹配、`.` 通配符
- `*`（零或多）、`+`（一或多）、`?`（零或一）量词
- `|` 分支、`()` 分组
- `[]` 字符类、`[^ ]` 否定类
- `[a-z]`、`[0-9]`、`[A-Z]` 字符范围
- `\d`、`\D`、`\w`、`\W`、`\s`、`\S` 转义
- `{n}`、`{n,}`、`{n,m}` 计数重复
- `^`、`$` 锚点

### API
- `matches(input)` — 完整匹配
- `find(input)` — 首次匹配位置
- `find_all(input)` — 所有匹配位置
- `replace(input, replacement)` — 替换所有匹配
- `split(input)` — 按匹配分割
- `count(input)` — 匹配计数
- `match_result(input)` — 详细匹配结果
